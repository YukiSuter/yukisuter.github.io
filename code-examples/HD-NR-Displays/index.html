The following was made as a small project to improve my JS and html skills. It takes any station in mainland Britain (England, Wales, Scotland) and uses an API to fetch information on that station's departures.

It is made to resemble the departure boards that are installed at London Victoria's Southern Railway side of the station.

It is basic, lacks more information to fully resemble the real deal, and also supports only a single page of calling stations.

No further updates will be made.
